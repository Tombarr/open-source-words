WebRTC Experiments & Demos It is a repository of uniquely experimented WebRTC demos; written by Muaz Khan! No special requirement! Just WebRTC compatible web-browser (e.g. chrome/firefox/opera on desktop/android) These demos/experiments are mostly client-side; i.e. no server installation needed! You can use all these demos in PHP/Python/Ruby/ASP.NET/etc. since they are only relying on JavaScript and 3rd party services! How to use? Each demo has a unique directory. Simply download that directory, upload in your webserver and use it; and itll work! You dont need to modify any single line to use it.
